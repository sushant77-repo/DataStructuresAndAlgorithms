**Leetcode 1092 - Shortest Common SuperSequence**

https://leetcode.com/problems/shortest-common-supersequence

Given two strings, return the shortest common supersequence.

Common supersequence is that string that contains both strings as subsequence.

E.g. s1 = “AGGTAB”

s2 = “GXTXAYB”

Ans → “AGGXTXAYB”
