**Leetcode 132 - Palindrome Partitioning II**

https://leetcode.com/problems/palindrome-partitioning-ii/description/

Given a string s, find minimum number of partitions required such that every partitioned substring is a palindrome.

e.g. s = “abcbx”

Ans → 2 → Two partitions will divide the string in “a”, “bcb”, “x” which are all palindromic.

Another example s = “nitin”

Ans → 0 →  String itself is a palindrome
