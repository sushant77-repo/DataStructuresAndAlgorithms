**Leetcode 322 - Coin Change** 

https://leetcode.com/problems/coin-change/

Given unlimited supply of coins of denominations from coin array,
find the minimum number of coins required to make the given sum.

e.g. arr[] = {1, 2, 3};

sum = 5

ans → 2 → 2 + 3 = 5
(we can make 5 using other combinations but this has min number of coins)
