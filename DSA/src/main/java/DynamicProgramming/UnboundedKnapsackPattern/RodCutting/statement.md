Given a rod of length L, we can cut it in as many parts as we want. Also given is length array and price array which gives price for the length of same index from length array. Cut in a way to maximize the total price. Return max price.

This is unbounded knapsack problem asked in different words.

e.g. len[] = {1, 2, 3, 4, 5}

price[] = {2, 8, 12, 14, 16}

L = 5;

ans  → 20 → we can cut it as 2 and 3 lengths.
